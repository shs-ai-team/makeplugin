Display a random quote with a button to show a new one. Works via shortcode or widget and matches your theme.

== Description ==
Random Quotes with Button displays a simple, theme-friendly quote block and a "New Quote" button that swaps to another random quote without reloading the page. Place it anywhere using the shortcode or the included widget. Ships with a small, varied set of starter quotes. Accessible markup and mobile-friendly by default.

== Usage ==
- Shortcode: [random_quotes_with_button]
- Widget: Appearance → Widgets → Random Quotes with Button

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the shortcode to posts/pages or use the widget in your sidebar/footer.

== Frequently Asked Questions ==
= Can I add my own quotes? =
Not in this version. The plugin ships with a built-in set of quotes.
